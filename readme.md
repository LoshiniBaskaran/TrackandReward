# Track and Reward

An application to log hours and track on initiatives taken and allow peers to recognize and applaud the contributions made.

## TechStack

This project is created with :
* Front-end
    * HTML
    * CSS
    * React
* Back-end
    * Nodejs - Express
* DB
    * MySQL
